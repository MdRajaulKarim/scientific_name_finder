To easily change colors in the future, just update the CSS variables at the top of `style.css`:

```css
    --red-dark: #b71c1c;         /* Change hover/active */
    --red-light: #ffebee;        /* Change light backgrounds */
    --bg: #F9F9F9;               /* Change page background */
    --text: #1a1a1a;             /* Change text color */
}
```
